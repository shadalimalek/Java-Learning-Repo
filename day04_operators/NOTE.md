*1. Arithmetic Operators in Java*

| Operator | Meaning             | Example (`a = 10`, `b = 3`) | Result |
| -------- | ------------------- | --------------------------- | ------ |
| `+`      | Addition            | `a + b`                     | `13`   |
| `-`      | Subtraction         | `a - b`                     | `7`    |
| `*`      | Multiplication      | `a * b`                     | `30`   |
| `/`      | Division            | `a / b`                     | `3`    |
| `%`      | Modulus (remainder) | `a % b`                     | `1`    |
